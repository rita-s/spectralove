# SPECTRA Love

**SPECTRA Love** is a dating app built around personality science, communication-style matching, and real-time conversation coaching.

It presents the core product idea through a premium single-page website built with plain HTML, CSS, and JavaScript.

---

## Overview

SPECTRA Love is  more than a swipe app.

It focuses on:

- **7 personality dimensions**
- **communication-style matching**
- **conversation coaching**
- **topic progression across relationship stages**
- **practice mode for improving conversation skills**

The page is designed to communicate the product vision, explain the matching logic, and collect early waitlist interest.
